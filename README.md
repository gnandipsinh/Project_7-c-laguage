# 🧮 C Calculator Program (Menu Driven)

## 📌 Description

This is a simple **menu-driven calculator program in C** that performs basic arithmetic operations using functions.

The program allows the user to:

* Add two numbers
* Subtract two numbers
* Multiply two numbers
* Divide two numbers
* Find modulus

It runs continuously until the user chooses to exit.

---

## ⚙️ Features

✔ Uses functions for each operation
✔ Menu-driven interface
✔ Infinite loop until user exits
✔ Simple and easy to understand

---

## 🧾 Operations Included

| Choice | Operation          |
| ------ | ------------------ |
| 1      | Addition (+)       |
| 2      | Subtraction (-)    |
| 3      | Multiplication (*) |
| 4      | Division (/)       |
| 5      | Modulus (%)        |
| 0      | Exit               |

---

## ▶️ How to Run

1. Save the file as:

```
calculator.c
```

2. Compile the program:

```
gcc calculator.c -o calculator
```

3. Run the program:

```
./calculator
```

---

## 💡 Example Output

```
--- MENU ---
Press 1 for +
Press 2 for -
Press 3 for *
Press 4 for /
Press 5 for %
Press 0 for Exit

Enter your choice: 1
Enter first number: 10
Enter second number: 5
Addition of 10 and 5 is 15
```

---

## ⚠️ Note

* Make sure **second number is not 0** while using division or modulus to avoid runtime errors.
